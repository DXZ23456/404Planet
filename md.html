项目介绍

404Planet 是一个探索数字“迷失世界”和错误状态概念的创意项目。

主要功能

  • ✨ 交互式网页界面
  • 📱 响应式设计
  • 🎨 可自定义主题
  • 🔗 开源项目

技术栈

  • 主要语言：HTML
  • 支持：CSS、JavaScript
  • 许可证：Apache License 2.0

快速开始

克隆仓库：

git clone https://github.com/DXZ23456/404Planet.git
cd 404Planet

在浏览器中打开 index.html 或启动本地服务器：

python -m http.server 8000
